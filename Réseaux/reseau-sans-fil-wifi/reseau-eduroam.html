Réseau Eduroam

  • Accueil   /  
  • Réseau Eduroam

L’académie de Nancy-Metz, membre de la communauté Eduroam, propose actuellement ce service sur les sites des DSDEN (pour les 4 départements) et du Rectorat.

Les utilisateurs d’un établissement membre d’Eduroam disposeront d’un accès sécurisé à l’Internet depuis tous les autres établissements membres en utilisant les paramètres d’authentification (identifiant et mot de passe) de leur établissement d’origine.

Pour plus d’informations et voir la liste des établissements membres, consultez le site Eduroam

Exemple d’utilisation

Vous êtes un personnel de l’académie de Nancy-Metz en mission sur un site de l’université de Lyon qui propose le réseau wifi “Eduroam” et vous souhaitez accéder à votre messagerie professionnelle : vous obtiendrez un accès internet en vous connectant à ce réseau wifi avec vos identifiants eduroam académiques.

Prérequis à la connexion

L’utilisation du service Eduroam nécessite de connaître votre adresse de messagerie académique (en @ac-nancy-metz.fr) et le mot de passe associé.

Pour rappel, l’adresse de messagerie est composée du prénom suivi du nom séparé par un point et éventuellement suivi d’un chiffre permettant de distinguer les homonymes. Il n’est pas possible d’utiliser une adresse fonctionnelle (celle d’un service ou d’un établissement par exemple).

  • L’identifiant Eduroam est de la forme : prenom.nom@ac-nancy-metz.fr
  • Le mot de passe est : votre mot de passe académique, vous ne pouvez pas utiliser votre clé OTP pour ce service

Connexion au réseau

Configuration automatique

Scannez ce QR code ou téléchargez l’utilitaire de configuration automatique à cette adresse : https://cat.eduroam.org/?idp=5589

Attention : Pour les appareils Android, il est nécessaire d’installer au préalable l’application « eduroam CAT » disponible sur GooglePlay.

Lancez l’exécutable que vous venez de télécharger. Puis connectez-vous avec votre email académique :

Configuration manuelle

Consulter les documentations de votre système d’exploitation, les caractéristiques pour l’accès Eduroam étant :

  • SSID : eduroam
  • Securité : 802.1x EAP (WPA2 Entreprise)
  • Méthode EAP : TTLS
  • Authentification étape 2 : PAP
  • Certificat CA : Utiliser les certificats du système
  • Etat du certificat en ligne : Ne pas valider
  • Domaine : ac-nancy-metz.fr
  • Identité : votre adresse mail académique (@ac-nancy-metz.fr)
  • Identité anonyme : anonymous@ac-nancy-metz.fr
  • Mot de passe : Votre mot de passe académique

Caractéristiques techniques

  • L’authentification réseau est obligatoire ;
  • Le chiffrement des données est activé ;
  • Liste des protocoles autorisés : HTTP (80/tcp), HTTPS (443/tcp), POPS (995/tcp), IMAPS (993/tcp), SMTPS (465/tcp) ;
  • Les protocoles réseaux nécessaires au VPN sont autorisés : SP, AH, GRE, CISCO Anyconnect (10000/tcp).

Informations complémentaires

  • L’usage du réseau WiFi implique le respect de la charte informatique ;
  • Le trafic vers l’internet est filtré, certains services ou URLs sont interdits (risques pour la sécurité, contenus illégaux, dégradation du service WiFi…) ;
  • Conformément à la réglementation en vigueur, toutes les connexions effectuées depuis les réseaux sans-fil sont conservées et pourront être transmises aux autorités judiciaires à leur demande ;
  • Les ressources du réseau interne académique (serveur bureautique, applications internes, imprimantes…) ne sont pas accessibles depuis ce réseau WiFi ;
  • Le réseau Eduroam est disponible tous les jours 24h/24.
